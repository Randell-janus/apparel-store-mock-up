# Mock-up Apparel Store

Shopping Cart implementation with **React** and **Firebase**

- https://apparel-store-mock-up.vercel.app/

## Technologies Used

### Frontend

- React
- TailwindCSS
- HeadlessUI

### Backend/Database

- Firebase
- Cloud Firestore

### State Management

- React Context API
